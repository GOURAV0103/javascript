# JavaScript DOM Projects

This repository contains a collection of beginner-friendly **JavaScript DOM practice projects**.  
These projects help in understanding how to manipulate the DOM, handle events, and build small interactive web applications using **HTML, CSS, and JavaScript**.

---

## 📂 Projects Included

### 06_DOM
A basic project to understand **Document Object Model (DOM)** concepts like selecting elements, changing content, and handling events.

### 07_color_changer
A simple project where users can change the **background color of the page** by clicking different buttons.  
This project demonstrates **event listeners and DOM manipulation**.

### 08_BMI_Calculator
A **Body Mass Index (BMI) Calculator** where users can enter their height and weight to calculate their BMI.  
It also shows whether the BMI is underweight, normal, or overweight.

### 09_Digital_Lock
A simple **Digital Lock interface** built with JavaScript that simulates entering a password or code.

### 10_Guess_number
A fun **Number Guessing Game** where the user tries to guess a randomly generated number within a limited number of attempts.
